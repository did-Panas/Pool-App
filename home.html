faq

Who can submit readings?
Anyone approved by the superintendent and/or property manager can submit readings. This can include security guards, and pool maintenance contractors.

How easy is it to submit a reading?
Submitting a reading with Prompt Pools is extremely easy and simple. The app is designed for user-friendliness, ensuring that anyone can quickly and efficiently submit chemical readings.

Can I get assistance with pool maintenance questions through Prompt Pools?
Yes, Prompt Pools features a virtual assistant that allows pool operators to seek guidance on pool maintenance queries. Whether it's adjusting pH levels or troubleshooting other chemical imbalances, the virtual assistant is available to provide helpful advice.

How long are readings stored in Prompt Pools?
Prompt Pools stores a year's worth of readings, allowing users to access historical data for reference and compliance purposes.

Does Prompt Pools facilitate monthly inspections?
Yes, Prompt Pools includes features to help users perform monthly inspections, which are required by law for pool maintenance and safety standards.

Are submitted readings tamper-proof?
While submitted readings can be edited, the original result is securely recorded and preserved within Prompt Pools. This ensures transparency and integrity, as the original data can be presented to health inspectors if required.

Is Prompt Pools free to use?
Yes, Prompt Pools is entirely free to use, eliminating the need for purchasing expensive log books and reducing operational costs associated with traditional paper-based methods.

Does Prompt Pools incorporate geolocation tracking?
Yes, Prompt Pools utilizes geolocation tracking to ensure that users must be near the pool area to submit readings, enhancing accuracy and compliance.

Can I print or email daily logs from Prompt Pools?
Yes, Prompt Pools allows users to conveniently print or email daily logs, providing flexibility and convenience in sharing important pool maintenance information.

How can I get started with Prompt Pools?
Getting started with Prompt Pools is easy! Simply book a demo with us to set up your account and begin submitting chemical readings for your pool.
